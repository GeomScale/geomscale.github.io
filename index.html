Sampling from multidimensional distributions is a fundamental operation that plays a crucial role across sciences including modern machine learning and data science. An impressive number of important problems such as optimization and integration can be efficiently solved via sampling.

GeomScale is a research and development project aiming in scalable algorithms to solve problems in these domains.

Currently, the main library is volesti, a generic open source C++ library, with R and Python interfaces, for sampling and volume computation in high dimensions. Moreover, volesti provides functionality for high-dimensional copula estimation that is useful in modelling financial crises.
